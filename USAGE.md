# 查重数据生成工具使用指南

`generate_dedup_data.py` 是一个独立工具，用于快速将候选人数据转换为 Excel 查重文件。

---

## 🚀 快速开始

### 方式 1: 从剪贴板读取（最快捷！⚡）

```bash
# 1. 在 Excel 中选择数据 → Cmd+C
# 2. 运行工具
python3 generate_dedup_data.py
# 3. 选择 1 (剪贴板) → 回车 → 完成！
```

### 方式 2: 从文件读取

```bash
# 1. 准备 .txt 文件（制表符分隔）
# 2. 运行工具
python3 generate_dedup_data.py
# 3. 选择 2 → 输入文件名 → 完成！
```

### 方式 3: 交互式输入

```bash
python3 generate_dedup_data.py
# 选择 3 → 逐行输入 → 空行结束
```

---

## 📝 数据格式

### 基本格式（制表符分隔）

```
姓名	公司	职位	电话	在职时间
```

**示例：**
```
张先生	某科技公司	产品经理	138****1234	23/5-Present
李女士	某AI公司	算法工程师	139****5678	24/2-Present
```

### 带分类格式（推荐）

```
分类A
张先生	某科技公司	产品经理	138****1234	23/5-Present
李女士	某AI公司	算法工程师	139****5678	24/2-Present

分类B
王先生	某互联网公司	运营经理	136****9012	20/7-Present
```

> 💡 **提示**: 从 Excel 复制时，制表符会自动保留！

---

## 📁 输出位置

生成的 Excel 文件保存到 `./data/` 目录，`main.py` 会自动加载进行查重。

---

## 🔄 与主脚本集成

1. 运行 `generate_dedup_data.py` 生成查重 Excel
2. 文件自动保存到 `./data/`
3. 运行 `main.py` 时自动加载，跳过重复候选人

---

## ❓ 常见问题

**Q: 剪贴板读取失败？**  
A: 确保已复制数据，或使用文件方式。

**Q: 数据没有正确解析？**  
A: 检查字段是否用制表符或多个空格分隔。

**Q: 生成的文件在哪里？**  
A: 在 `./data/` 目录下。
